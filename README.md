
# CrispyChannels

Projekt ktory tworzy kanaly np na spawnie, dzieki ktorym rozdzielamy spawny na kilka serwerow, co powoduje lepsza wydajnosc, mniej graczy na serwerze. Projekt jest oparty o protobufa oraz kryoneta, co daje w polaczenie niezle zwierze. Benchmarki pojawia sie kiedys
Mile widziane PR oraz ISSUE (nie wystapia hehe)
